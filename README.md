# FORK

This is a Jobwall for alumni account.
Here you can find favorite jobs and you can mark them as favorite.

![jobwall-alumni](./images/jobwall-alumni.png)

If you click on the favorite tab, you can see your favorite jobs.
![favoritejobs](./images/favoritejobstab.png)

You can see the details of a job
![jobdetail](./images/jobdetailpopup.png)


This is a Jobwall for business account.
Here you can post jobs.
![jobwall-business](./images/jobwall-business.png)

If you click the edit button, you can edit the post.
![jobeditpopup](./images/jobeditpopup.png)

If you click the delete button, you can delete the post.
![jobdeletepopup](./images/jobdeletepopup.png)
